我會這些:

JavaScript

JavaScript可以讓你在網頁中實現出複雜功能的程式語言.
我目前最精通此語言，2022/9/3創建了從一開始到現在的DiscordBot!

CSS

CSS 既非標準程式語言，也不是標記語言。
它能讓你在 HTML 文件中的元素上套用不同的頁面樣式。

HTML

HTML是打造網頁的基石。它表述並定義網頁的內容
伴隨 HTML 而來的技術還有描述網頁外觀(CSS)及功能性的程式語言(JavaScript)。

Discord.js

discord.js 是一個功能強大的 Node.js 模塊。可用性、一致性和性能是 discord.js 的特點。
它對 Discord API 的覆蓋率也接近 100%。它在到達 API 後不久收到新的 Discord 功能。
